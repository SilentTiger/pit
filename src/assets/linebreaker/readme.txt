这个目录下的代码用来处理字符串的断行逻辑，这段逻辑非常重要，除非你知道在干嘛，否则不要瞎改！！！

这个目录的下的代码主要来自 https://github.com/foliojs/linebreak 
没有直接通过 npm 安装这个第三方库是因为它本身是为 node 写的，依赖 fs 模块，无法在浏览器环境中使用

我做了一点点修改，主要就是把用 fs 读取 classes.trie 文件的逻辑去掉，改为直接把读出来的数据硬编码到代码文件中
这样就可以在浏览器中运行了

另外，这个第三方的仓库有几个依赖项 base64-js、unicode-trie，这两个依赖项一定要记得装

最后，改造之后的 linebreak 并不具有和原来完全一致的功能，目前仅仅是断行算法可用，其他功能需要参考原来的代码及文档

Good luck
